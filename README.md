# Projeto Vai na Web (Site Ritmo da batida)

Como parte do desafio proposto pelo curso "Vai na Web", criei um website, que consiste na criação de um site abordando música, e-books e vídeos. O objetivo é criar uma interface simples, porém rica em elementos CSS, especialmente utilizando propriedades do Flexbox para gerenciar espaçamentos. O site é composto por um menu principal, seguido por uma seção de conteúdo principal e um submenu. Na parte principal, incluí uma seção de fotos e um link decorativo que direciona para outras páginas relacionadas ao tema. O rodapé contém links para redes sociais e um menu de navegação.
É importante destacar que o site ainda não é responsivo, portanto, para uma experiência mais agradável, recomenda-se visualizá-lo nas resoluções de tela de 1285 x 963 pixels e 1300 x 963 pixels.

## Tecnologias Utilizadas
- HTML5
- CSS3
- CodeSandbox
